# Cadastro de Pessoas

Projeto de estudo para desenvolvimento de uma aplicação web completa.

Objetivo do projeto:

- Aprender programação em C#
- Construir uma API .NET
- Implementar cadastro de pessoas com validações
- Criar interface web para cadastro
- Publicar aplicação online

Funcionalidades previstas:

- Cadastro de Pessoa Física e Jurídica
- Validação de CPF e CNPJ
- Controle de status (ativo, inativo, bloqueado)
- Histórico de alterações
- API REST para integração
