# Visão do Produto

Projeto: Cadastro de Pessoas

Objetivo:
Criar uma aplicação web para gerenciamento de cadastro de pessoas com governança de dados.

Problemas que o sistema resolve:

- Cadastros duplicados
- Falta de validação de CPF/CNPJ
- Falta de controle de alterações
- Dados inconsistentes

Funcionalidades principais:

1. Cadastro de Pessoa Física e Jurídica
2. Validação de CPF e CNPJ
3. Controle de status (ativo, inativo, bloqueado)
4. Registro de histórico de alterações
5. API REST para integração com outros sistemas

Tecnologias previstas:

Backend:
- C#
- .NET Web API

Frontend:
- Angular

Banco de dados:
- PostgreSQL
